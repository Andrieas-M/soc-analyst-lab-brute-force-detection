# SOC Analyst Lab: Brute Force Attack Detection

## Objective
Simulate and detect a brute force attack using system logs.

## Tools Used
- Windows Event Viewer / Linux logs
- Basic command line

## Skills Demonstrated
- Log analysis
- Threat detection
- Incident response

## Scenario
Multiple failed login attempts were detected on a system.

## Outcome
Identified suspicious IP and recommended blocking actions.
